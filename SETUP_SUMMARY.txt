🚀 QUICK START (15 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Get Client ID
   → https://console.cloud.google.com/
   → Credentials → OAuth 2.0 Client ID → Web
   → Copy the Client ID (format: 123...apps.googleusercontent.com)

2. Add to Config
   → cd C:\Users\User.J1-ALPHA-PENS\pbsys
   → ./google-signin-setup.sh "YOUR_CLIENT_ID"

3. Verify Setup
   → php verify-google-signin.php
   (Should show all green ✓ checks)

4. Test Locally
   → ./dev.sh
   → http://localhost:8000/auth/login.php?dev=1

5. Deploy to Hestia
   → rclone copy includes/secrets.php \
       hestia:web/neovtrack.uitm.edu.my/public_html/includes/ \
       --sftp-disable-hashcheck

6. Test Live
   → https://neovtrack.uitm.edu.my/auth/login.php
   (Should show the Google button)

7. Sign In
   → Add yourself to admin_allowlist (SQL or admin panel)
   → Click the Google button
   → Redirect to dashboard

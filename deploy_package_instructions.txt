Deployment package for search_api.php

Files included:
 - search_api.php (place in webroot alongside searchCar.php)

Instructions for Hestia File Manager:
1. Log in to Hestia control panel and open File Manager for the site (neovtrack.uitm.edu.my).
2. Navigate to public_html (or the webroot where searchCar.php resides).
3. Click Upload and choose the file search_api.php from this package.
4. Set file permissions to 644 and owner/group to the web user (usually nobody/www-data or the hosting user).
5. Test in browser: https://neovtrack.uitm.edu.my/search_api.php (or use http if site uses http).
6. If 404 occurs, check file placement and virtual host DocumentRoot. Alternatively, upload to the same directory as other site PHP files.

SFTP instructions:
1. Connect via SFTP to the server (host, username, password or key).
2. Upload search_api.php to the public_html root.
3. Set permissions as above.
4. Visit the URL to confirm.

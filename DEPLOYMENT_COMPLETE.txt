================================================================================
                         TROUBLESHOOTING
================================================================================

Problem: 404 Not Found
Solution:
  - Verify files are in public_html directory
  - Check file names are exactly as listed (case-sensitive)
  - Verify file permissions are 644
  - Check virtual host DocumentRoot configuration

Problem: 403 Forbidden
Solution:
  - Check file permissions (should be 644)
  - Verify owner/group is web user
  - Check firewall rules

Problem: Database Connection Error
Solution:
  - Verify connect.php is in same directory
  - Check database credentials
  - Test connection: php test_connection.php

Problem: API Returns 403 (Access Denied)
Solution:
  - Ensure logged in as admin
  - Verify userid <= 10 (superadmin requirement)
  - Check session cookie is set

Problem: Statistics Don't Display
Solution:
  - Check database has vehicle records
  - Verify tables exist in database
  - Clear browser cache
  - Check browser console for errors
